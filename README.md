# 音乐播放平台系统 python1100

## 项目概述

音乐播放平台系统是基于Python语言开发，结合Vue2和Django框架，使用MySQL数据库的音乐分享与播放平台。

## 技术栈

- 后端：Python 3.8 + Django
- 前端：Vue2
- 数据库：MySQL 5.7

- 普通用户
